Project Summary

The Campus Booking System is a SwiftUI iOS prototype enabling university students to book study rooms, equipment, and tutoring sessions. Phase 1 focuses on requirements documentation, prototype architecture, and automated tooling evidence across seven epics: User Accounts, Tutor Booking, Room Booking, Equipment Booking, Notifications, Admin Functionality, and Ratings & Feedback. The deliverable is a focused iOS mockup built with MVVM architecture in Swift / SwiftUI targeting iOS 17+, demonstrating core user flows and system design rather than a production deployment.
